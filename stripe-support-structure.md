# Stripe Support Page Structure Analysis

## Key Design Elements

### Layout
- Clean white background
- Simple header with logo + search bar + navigation links
- Main content area on left (wider)
- Sidebar on right with "Popular topics" tags

### Typography
- Large, bold title (h1)
- Topic tags below title (2FA, Account, Dashboard)
- Clear section headings (h3 style)
- Body text in regular weight, good line spacing

### Content Structure
- Title at top
- Category tags immediately below title
- Content divided into clear sections with headings
- Each section is a question/answer format
- Bullet points for lists
- Inline links for actions
- "Did this answer your question?" feedback at bottom
- Related articles section

### Colour Scheme
- White background
- Dark text (near black)
- Blue links
- Light grey for secondary elements
- Minimal use of colour - only for links and CTAs

### Sidebar
- "Popular topics" heading
- Tag-style links for categories
- "Contact support" CTA at bottom

## Applying to CCMS Help Center

1. Simplify the layout - remove gradient backgrounds
2. Use white/light grey background
3. Single primary colour (#1F7AE0) for links and CTAs
4. Clean typography with clear hierarchy
5. Category tags below article title
6. Question/answer format for content sections
7. Related articles at bottom
8. Simple feedback section
